APP 的结构和行为

App​

iOS 14

表示 APP 结构和行为的类型。

import SwiftUI

@main
struct MyApp: App {
    var body: some Scene {
        WindowGroup {
            Text("Hello, world!")
        }
    }
}

您还可以创建自定义场景。 要在应用程序中包含多个场景，请添加@SceneBuilder 属性添加到正文。

@main
struct Mail: App {
    @SceneBuilder var body: some Scene {
        WindowGroup {
            MailViewer()
        }
        Settings {
            SettingsView()
        }
    }
}
